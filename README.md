# Global Solution 2026.1

## Sistema Inteligente de Monitoramento Energético para Missão Espacial Experimental

## Objetivo

Este projeto foi desenvolvido para a Global Solution 2026.1 do curso de Ciência da Computação.

A proposta consiste na criação de um sistema inteligente capaz de monitorar condições operacionais de uma missão espacial experimental, analisando dados simulados relacionados à temperatura, energia, comunicação e status dos módulos da missão.

O sistema é capaz de identificar situações críticas, gerar alertas automáticos e executar ações corretivas de forma autônoma, contribuindo para a segurança operacional da missão.

## Funcionalidades

* Monitoramento de temperatura
* Monitoramento energético
* Verificação do status de comunicação
* Monitoramento dos módulos da missão
* Geração automática de alertas
* Tomada de decisão automatizada
* Simulação de dados em tempo real

## Tecnologias Utilizadas

* Python 3
* Programação Estruturada
* Algoritmos de Tomada de Decisão
* Simulação de Dados

## Funcionamento

O sistema gera dados simulados para representar o estado operacional de uma missão espacial.

Os dados analisados são:

* Temperatura do sistema
* Nível de energia disponível
* Status da comunicação
* Estado dos módulos da missão

Quando condições críticas são detectadas, o sistema executa ações automáticas como:

* Ativação de modo de economia de energia
* Acionamento de sistema de resfriamento
* Tentativa de reconexão da comunicação
* Isolamento de módulos críticos

## Exemplo de Saída

Temperatura: 37°C

Energia: 14%

Comunicação: ONLINE

Status do Módulo: CRÍTICO

ALERTAS:

* Energia crítica
* Módulo em estado crítico

AÇÕES:

* Economia de energia ativada
* Isolamento do módulo crítico

## Conclusão

O projeto demonstra a aplicação de conceitos de monitoramento inteligente, análise de dados, automação de decisões e sustentabilidade energética em ambientes espaciais experimentais.

A solução contribui para aumentar a eficiência operacional da missão e reduzir riscos decorrentes de falhas críticas.
